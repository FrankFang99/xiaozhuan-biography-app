# 小传 - 设计系统

## 核心隐喻

「信件」→「书籍」→「星空」

每一次对话都是一封被珍藏的信，信夹是用户的记忆容器，AI将信件整理成一本书，所有老人的故事汇聚成一片星空。

## 组件规范

### 信封组件
- 尺寸：宽280px x 高180px
- 颜色：#F5E6C8 + #C73E3A封口
- 阴影：0 4px 12px rgba(0,0,0,0.1)
- 状态：收起/展开

### 信纸组件
- 尺寸：自适应宽度
- 颜色：#FEFCF3
- 纹理：细微纸纹背景
- 边框：1px #E8DCC8

### 书籍组件
- 尺寸：宽300px x 高420px
- 颜色：封面#F5E6C8，内页#FEFCF3
- 阴影：0 12px 40px rgba(0,0,0,0.25)
- 状态：封面/翻开/关闭

### 星星组件
- 尺寸：4px-16px
- 颜色：#D4A853 / #F4D06F
- 动画：呼吸闪烁

## 页面结构

1. **星空首页** - 深邃星空背景，品牌主张
2. **目标设定** - 选择自传周期
3. **信夹** - 展示已收集的信件
4. **写信** - 对话收集故事
5. **书籍** - 成书展示
