## 1. Product Overview
小传 - 为每一个平凡人生写传记的AI应用，通过与老人对话收集人生故事，整理成传记，并实现星空纪念概念。
- 目标用户：50-80岁老年人及他们的子女
- 核心价值：让普通人的故事被看见、被记住、被传承

## 2. Core Features

### 2.1 Feature Module
1. **展示首页**：电影级星空背景，品牌主张，CTA入口
2. **问题陈述**：普通人传记缺失的痛点展示
3. **产品流程**：5步流程展示（首页→目标设定→信夹→写信→成书）
4. **星空纪念**：纪念概念展示，动态星星效果
5. **技术方案**：6项核心技术介绍
6. **团队介绍**：团队成员信息

### 2.2 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 展示首页 | Hero区 | 深靛蓝星空背景，150颗呼吸闪烁星星，品牌大字"小传"，金色光晕效果 |
| 问题陈述 | 引言区 | 痛点描述，金句引用，优雅排版 |
| 产品流程 | 手机Mockup | 5个精致手机界面，含状态栏、notch、真实UI内容 |
| 星空纪念 | 动态星空 | 15颗不同状态星星（活跃/暗淡/最亮），生日脉冲动画 |
| 技术方案 | 卡片网格 | 6项技术介绍，悬停动效 |
| 团队介绍 | 成员卡片 | 头像、姓名、职位 |

## 3. Core Process
用户打开页面 → 观看星空首页 → 了解问题 → 查看产品流程 → 理解星空纪念概念 → 了解技术方案 → 认识团队

## 4. User Interface Design
### 4.1 Design Style
- **主色调**：深靛蓝(#050A14)、星空蓝(#0A1628)
- **点缀色**：星光金(#D4A853)、星光亮(#F4D06F)、信封红(#C73E3A)
- **信纸色**：暖米色(#F5E6C8)、纸张白(#FEFCF3)
- **字体**：Noto Serif SC（思源宋体），优雅衬线字体
- **布局**：杂志式排版，大量留白，不对称设计
- **动效**：滚动触发渐入、星星呼吸闪烁、悬停微交互

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| Hero区 | 星空背景 | 150颗分层星星，渐变背景，品牌大字8rem，金色光晕 |
| 问题陈述 | 文字排版 | 大号引言，中文引号装饰，字间距优化 |
| 产品流程 | 手机Mockup | iPhone风格notch，状态栏，真实UI内容，阴影质感 |
| 星空纪念 | 动态星星 | 三种状态星星，脉冲动画，径向渐变容器 |
| 技术方案 | 卡片 | 玻璃态效果，边框发光，悬停上浮 |

### 4.3 Responsiveness
- 桌面端：多列网格布局
- 移动端：单列堆叠，响应式字体和间距
